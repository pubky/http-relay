# File size and folder structure

A file should be max 400 lines. In rare circumstances, 600 lines is the absolute max.
**If a file exceeds this limit, split it** - but use judgment: only split during non-trivial changes, not for small fixes like typos.

Think about the folder/file tree as an easy search index to find knowledge. If everything would be
in one file, things would be hard to find. A folder/file tree helps organize knowledge
so the person after you can find it again easily.
